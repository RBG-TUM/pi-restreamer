# pi-restreamer
A Dockerfile installing NGINX, nginx-rtmp-module and FFmpeg from source and starting a rtmp stream from capture cards that can be consumed by TUM-Live

* Nginx 1.21.0 (Mainline version compiled from source)
* nginx-rtmp-module 1.2.2 (compiled from source)
* ffmpeg 4.4 (compiled from source)
